Quick Start

Two lines of HTML. That's all it takes.

HTML
<!-- Step 1: Place widget divs anywhere on your page -->
<div data-unicodefyi="entity" data-slug="example" data-theme="light"></div>
<div data-unicodefyi="compare" data-slugs="item-a,item-b"></div>
<div data-unicodefyi="search" data-placeholder="Search Characters..."></div>

<!-- Step 2: Load the embed script once, anywhere -->
<script src="https://cdn.jsdelivr.net/npm/unicodefyi-embed@1/dist/embed.min.js"></script>

Widget Options

Configure widgets with data-* attributes.

Attribute Values Default Description
data-unicodefyientity, compare, glossary, guide, search, tooltip, [tools]requiredWidget type to render
data-slugany slug—Entity slug from the UnicodeFYI database
data-themelight, dark, sepia, autolightVisual theme (auto follows OS preference)
data-styleVariantmodern, minimalmodernWidget design style
data-sizedefault, compact, largedefaultWidget size
data-slugsslug-a,slug-b—Comma-separated slugs for compare widget
data-placeholderany stringSearch...Search box placeholder text

Styles & Themes

2 design styles × 4 themes. Mix and match to fit your site.

modern

Clean lines, rounded corners, accent gradients. Ideal for product pages and marketing sites. Default style.

data-styleVariant="modern"
minimal

Subtle borders, flat design, no gradients. Ideal for documentation and reference sites.

data-styleVariant="minimal"
Light + Dark + Auto

Set explicitly or use auto to follow the visitor's OS preference (prefers-color-scheme).

data-theme="auto"
Sepia

Warm paper-tone background. Perfect for design blogs, typography articles, and creative content.

data-theme="sepia"

CDN Options

Multiple delivery options to fit your workflow.

jsDelivr CDN

Global CDN, auto-updates with npm. Recommended for most use cases.

HTML
<script src="https://cdn.jsdelivr.net/npm/unicodefyi-embed@1/dist/embed.min.js"></script>

npm Package

For Webpack, Vite, Rollup bundlers.

Shell
npm install unicodefyi-embed

// In your JS/TS:
import 'unicodefyi-embed';

Pinned Version

Lock to a specific version for production stability.

HTML
<script src="https://cdn.jsdelivr.net/npm/unicodefyi-embed@1.0.0/dist/embed.min.js"></script>
